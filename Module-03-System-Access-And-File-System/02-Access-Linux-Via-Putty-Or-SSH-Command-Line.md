# Access Linux via PuTTY or SSH Command Line

Linux systems can be accessed remotely using SSH
from both Windows and Linux operating systems.

## Using SSH from Linux or macOS
- Open a terminal
- Connect using the SSH command:
  ssh username@server_ip
- Enter the user password when prompted

## Using PuTTY on Windows
- Open PuTTY
- Enter the server IP address
- Select SSH as the connection type
- Click Open and authenticate with your credentials

SSH provides encrypted and secure remote access
and is the standard method for Linux administration.
