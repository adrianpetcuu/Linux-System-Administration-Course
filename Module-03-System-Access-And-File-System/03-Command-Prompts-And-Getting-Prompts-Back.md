# Command Prompts and Getting Prompts Back

The command prompt indicates that the system is ready
to accept user commands.

## Common prompts
- `$` regular user prompt
- `#` root user prompt

## Getting the prompt back
- Press Ctrl + C to stop a running command
- Press Enter if the terminal appears idle
- Use Ctrl + Z to suspend a process
- Type `fg` to resume a suspended process

Understanding prompts helps manage commands
and troubleshoot stuck terminals.
