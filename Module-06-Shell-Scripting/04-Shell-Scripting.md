# Shell Scripting

Shell scripting is the practice
of writing scripts using a shell
to automate tasks in Linux.

## What is a shell script
- A text file containing shell commands
- Executed sequentially by the shell
- Used for automation and system administration

## Script basics
- Script file usually ends with `.sh`
- Must start with a shebang:
```
#!/bin/bash
```

## Common elements
- Variables
- Conditional statements (if, else)
- Loops (for, while)
- Command substitution

## Example
```
#!/bin/bash
echo "Hello, Linux"
```

## Run a script
- Make it executable:
```
chmod +x script.sh
```
- Execute:
```
./script.sh
```

Shell scripting
is essential for automation,
administration, and DevOps tasks.
