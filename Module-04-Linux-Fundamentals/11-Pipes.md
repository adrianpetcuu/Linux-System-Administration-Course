# Pipes

Pipes allow the output of one command
to be used as input for another command.

## Pipe operator
- `|` â€“ connects commands together

## Basic usage
- `command1 | command2`
- Output of command1 becomes input of command2

## Common examples
- `ls | less`
- `ps aux | grep ssh`
- `cat file.txt | wc -l`

Pipes are essential
for combining simple commands into powerful workflows.
