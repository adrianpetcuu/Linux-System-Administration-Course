# wc – Text Processing Command

The `wc` command is used
to count lines, words, and characters
in files or command output.

## Common options
- `-l` – count lines
- `-w` – count words
- `-c` – count bytes
- `-m` – count characters

## Examples
- `wc file.txt`
- `wc -l file.txt`
- `ls | wc -l`
- `wc -w document.txt`

The `wc` command is commonly used
for quick statistics and reporting.
