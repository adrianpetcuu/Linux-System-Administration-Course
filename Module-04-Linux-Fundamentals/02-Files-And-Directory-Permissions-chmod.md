# Files and Directory Permissions (chmod)

Linux permissions control access
to files and directories.

## Permission types
- r (read) – view file or list directory
- w (write) – modify file or directory
- x (execute) – run file or access directory

## Permission levels
- User (u) – file owner
- Group (g) – group owner
- Others (o) – all other users

## Changing permissions
- Symbolic mode: chmod u+x file
- Numeric mode: chmod 755 file

Permissions are essential
for system security and access control.
