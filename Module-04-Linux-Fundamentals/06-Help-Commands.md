# Help Commands

Linux provides several built-in
ways to get help and documentation
for commands and system usage.

## Common help commands
- `man command` – display the manual page
- `command --help` – show command usage options
- `info command` – detailed documentation
- `apropos keyword` – search manual pages by keyword

## Examples
- `man ls`
- `ls --help`
- `apropos network`

Using help commands is essential
for learning and troubleshooting in Linux.
