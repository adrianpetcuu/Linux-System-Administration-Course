# Linux vs Windows Commands

Linux and Windows use different
command-line tools to perform
similar system tasks.

## Common command equivalents

| Task | Linux Command | Windows Command |
|-----|---------------|-----------------|
| List files | ls | dir |
| Change directory | cd | cd |
| Copy files | cp | copy |
| Move / rename | mv | move / ren |
| Delete files | rm | del |
| Create directory | mkdir | mkdir |
| Display file content | cat | type |
| Find text | grep | findstr |
| Show processes | ps | tasklist |
| Kill process | kill | taskkill |

Understanding command equivalents
helps users transition from Windows
to Linux environments.
