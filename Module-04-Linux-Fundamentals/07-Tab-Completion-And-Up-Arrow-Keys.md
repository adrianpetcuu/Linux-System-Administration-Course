# Tab Completion and Up Arrow Keys

Linux command-line provides keyboard features
that improve speed and efficiency.

## Tab completion
- Press `Tab` to auto-complete commands, files, or directories
- Press `Tab` twice to list multiple possible matches
- Reduces typing errors and saves time

## Up Arrow key
- Press the Up Arrow key to recall previous commands
- Allows quick reuse or editing of commands
- Useful for repetitive tasks

These features are essential
for efficient command-line navigation.
