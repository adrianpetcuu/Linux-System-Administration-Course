# Linux File Editor – vi

`vi` is a powerful and standard
text editor available on all
Linux and Unix systems.

## Modes in vi
- **Normal mode**: default mode
  for navigation and commands
- **Insert mode**: used to
  enter or edit text
- **Command mode**: used to
  save, quit, or execute actions

## Common vi commands
- `i` → insert before cursor
- `a` → append after cursor
- `o` → new line below
- `O` → new line above
- `Esc` → return to normal mode

## Saving and exiting
- `:w` → save file
- `:q` → quit
- `:wq` → save and quit
- `:q!` → quit without saving

`vi` is essential for system
administration and RHCSA exams.
