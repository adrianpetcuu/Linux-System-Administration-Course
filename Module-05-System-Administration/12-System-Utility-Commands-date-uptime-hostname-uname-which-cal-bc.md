# System Utility Commands

Linux provides system utility commands
to display system information
and perform basic calculations.

## date
- Displays or sets the system date and time
- Example: `date`
- Example: `date "+%Y-%m-%d"`

## uptime
- Shows how long the system has been running
- Displays load average
- Example: `uptime`

## hostname
- Displays or sets the system hostname
- Example: `hostname`
- Example: `hostnamectl set-hostname MyLinuxVM`

## uname
- Displays system and kernel information
- Common options:
  - `-a` – all system information
  - `-r` – kernel release
- Example: `uname -a`

## which
- Shows the full path of a command
- Example: `which ls`

## cal
- Displays a calendar
- Example: `cal`
- Example: `cal 2026`

## bc
- Command-line calculator
- Supports arithmetic operations
- Example:
  - `bc`
  - `5 + 10`

These system utilities
are commonly used for monitoring
and basic system interaction.
