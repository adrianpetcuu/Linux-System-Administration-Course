# top Command

The `top` command provides
a real-time view of running
processes and system resource usage.

## What top displays
- CPU usage
- Memory usage
- Running processes
- System load average

## Common keys in top
- `P` – sort by CPU usage
- `M` – sort by memory usage
- `k` – kill a process
- `r` – change process priority
- `q` – quit top

## Example
- `top`

The `top` command
is commonly used
to monitor system performance
and identify resource-intensive processes.
