# User Account Management

Linux provides commands
to create, modify, and remove
users and groups.

## User management commands
- `useradd` – create a new user account
- `usermod` – modify an existing user
- `userdel` – delete a user account

## Group management commands
- `groupadd` – create a new group
- `groupdel` – delete a group

## Common examples
- `useradd john` – create user john
- `passwd john` – set user password
- `usermod -aG wheel john` – add user to a group
- `userdel john` – remove user
- `groupadd developers` – create group
- `groupdel developers` – delete group

User and group management
is a core responsibility
of Linux system administrators.
