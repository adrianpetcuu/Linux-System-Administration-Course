# Finding System Architecture (arch)

Linux provides commands
to identify the system
hardware architecture.

## arch command
- Displays the machine architecture
- Useful for determining compatibility
- Example:
  - `arch`

## Common outputs
- `x86_64` – 64-bit Intel/AMD architecture
- `i686` / `i386` – 32-bit architecture
- `aarch64` – 64-bit ARM architecture

## Related commands
- `uname -m` – shows machine hardware name
- `lscpu` – detailed CPU architecture info

Knowing the system architecture
is important for installing
correct software packages.
